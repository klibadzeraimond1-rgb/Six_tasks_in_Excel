 დავალებების აღწერა და გამოყენებული ფუნქციები:

 ამოცანა 1: მონაცემთა იდენტიფიკაცია
* **შინაარსი:** უნიკალური კოდების შექმნა სტუდენტებისა და უნივერსიტეტებისთვის.
* **ინსტრუმენტი:** `CONCATENATE`, ტექსტური ფუნქციები.

 ამოცანა 2: სამუშაო პროცესების ვადების მართვა
* **შინაარსი:** პროექტების დაწყება-დასრულების თარიღების გამოთვლა უქმე დღეების გათვალისწინებით.
* **ინსტრუმენტი:** `NETWORKDAYS`, თარიღის ფორმატები.

 ამოცანა 3: ფინანსური რეპორტინგი
* **შინაარსი:** გაყიდვების, მომხმარებლების რაოდენობისა და ბონუსების კალკულაცია.
* **ინსტრუმენტი:** `IF` ფუნქცია, მათემატიკური ფორმულები.

 ამოცანა 4: ინვენტარიზაცია
* **შინაარსი:** მაღაზიებისა და პროდუქციის (ლეპტოპები, მონიტორები და ა.შ.) აღრიცხვა.
* **ინსტრუმენტი:** `VLOOKUP`, `SUMIF`.

 ამოცანა 5: პერსონალის მონაცემთა ბაზა
* **შინაარსი:** თანამშრომელთა კონტრაქტების, დაზღვევებისა და ხელფასების მართვა.
* **ინსტრუმენტი:** `Data Filtering`, `Conditional Formatting`.

 ამოცანა 6: მონაცემთა ვალიდაცია
* **შინაარსი:** პირადი ნომრებისა და ელ-ფოსტის სისწორის შემოწმება.
* **ინსტრუმენტი:** `Data Validation`, `Logical Tests`.
