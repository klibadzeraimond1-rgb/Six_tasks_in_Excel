## ­ЪЊѓ рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў:

### ­Ъћ╣ рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ 1: рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљ
* **рЃерЃўрЃюрЃљрЃљрЃарЃАрЃў:** рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃЎрЃЮрЃЊрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃБрЃюрЃўрЃЋрЃћрЃарЃАрЃўрЃбрЃћрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
* **рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃў:** `CONCATENATE`, рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў.

### ­Ъћ╣ рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ 2: рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃърЃарЃЮрЃфрЃћрЃАрЃћрЃЉрЃўрЃА рЃЋрЃљрЃЊрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ
* **рЃерЃўрЃюрЃљрЃљрЃарЃАрЃў:** рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ-рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃЌрЃљрЃарЃўрЃдрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃљ рЃБрЃЦрЃЏрЃћ рЃЊрЃдрЃћрЃћрЃЉрЃўрЃА рЃњрЃљрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЉрЃўрЃЌ.
* **рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃў:** `NETWORKDAYS`, рЃЌрЃљрЃарЃўрЃдрЃўрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃћрЃЉрЃў.

### ­Ъћ╣ рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ 3: рЃцрЃўрЃюрЃљрЃюрЃАрЃБрЃарЃў рЃарЃћрЃърЃЮрЃарЃбрЃўрЃюрЃњрЃў
* **рЃерЃўрЃюрЃљрЃљрЃарЃАрЃў:** рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА, рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЉрЃЮрЃюрЃБрЃАрЃћрЃЉрЃўрЃА рЃЎрЃљрЃџрЃЎрЃБрЃџрЃљрЃфрЃўрЃљ.
* **рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃў:** `IF` рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ, рЃЏрЃљрЃЌрЃћрЃЏрЃљрЃбрЃўрЃЎрЃБрЃарЃў рЃцрЃЮрЃарЃЏрЃБрЃџрЃћрЃЉрЃў.

### ­Ъћ╣ рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ 4: рЃўрЃюрЃЋрЃћрЃюрЃбрЃљрЃарЃўрЃќрЃљрЃфрЃўрЃљ
* **рЃерЃўрЃюрЃљрЃљрЃарЃАрЃў:** рЃЏрЃљрЃдрЃљрЃќрЃўрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃърЃарЃЮрЃЊрЃБрЃЦрЃфрЃўрЃўрЃА (рЃџрЃћрЃърЃбрЃЮрЃърЃћрЃЉрЃў, рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃћрЃЉрЃў рЃЊрЃљ рЃљ.рЃе.) рЃљрЃдрЃарЃўрЃфрЃ«рЃЋрЃљ.
* **рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃў:** `VLOOKUP`, `SUMIF`.

### ­Ъћ╣ рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ 5: рЃърЃћрЃарЃАрЃЮрЃюрЃљрЃџрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљ
* **рЃерЃўрЃюрЃљрЃљрЃарЃАрЃў:** рЃЌрЃљрЃюрЃљрЃЏрЃерЃарЃЮрЃЏрЃћрЃџрЃЌрЃљ рЃЎрЃЮрЃюрЃбрЃарЃљрЃЦрЃбрЃћрЃЉрЃўрЃА, рЃЊрЃљрЃќрЃдрЃЋрЃћрЃЋрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃ«рЃћрЃџрЃцрЃљрЃАрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ.
* **рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃў:** `Data Filtering`, `Conditional Formatting`.

### ­Ъћ╣ рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ 6: рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ
* **рЃерЃўрЃюрЃљрЃљрЃарЃАрЃў:** рЃърЃўрЃарЃљрЃЊрЃў рЃюрЃЮрЃЏрЃарЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃћрЃџ-рЃцрЃЮрЃАрЃбрЃўрЃА рЃАрЃўрЃАрЃгрЃЮрЃарЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ.
* **рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃў:** `Data Validation`, `Logical Tests`.
